GitHub Code Mining Agent Agentic RAG · v1

Purpose & outcome

Purpose: Turn a plain-English goal into evidence-backed insight from GitHub — the model decides which live GitHub operations to run (repo search, code search, README, commits, file reads), then synthesises an answer. This is Agentic AI (ReAct tool loop) plus Agentic RAG (retrieve real artifacts, then generate), not a generic chat reply.

Outcome: Answers you can trust for homework and demos — with owner/repo and paths grounded in tool output. Unauthenticated GitHub is rate-limited; for code search, add GITHUB_TOKEN on the server or paste a PAT below.

Stack alignment: FastAPI gateway → LangChain ReAct agent → Google Gemini · Reuses utilities/appGitHubCodeMining planner + contextual GitHub helpers.
