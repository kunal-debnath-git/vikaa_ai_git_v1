Vikaa.AI — Agent Workspace

Vikaa.AI is a personal AI engineering workspace — Multimodal Chat, Web Research, Gmail Intelligence, Databricks Integration, Synthetic Data Generation, and more, all connected through a single FastAPI + Supabase backbone.

Chat mode for open-ended thinking and exploration. Tool mode when a task needs its own focused interface. One platform — no context switching, no tab overload.

Execution is access-controlled — only invited users can trigger runs. A purposeful lab environment, not an open playground.

New capabilities added regularly. More connectors in progress.
